In-Screen Unit Generator

  • In-Screen units must be provided as one 100% x 100px Hype file, with breakpoint scenes inside
  • Edit all scenes within the Hype file to click out (in appropriate areas) with: openAd(hypeDocument, element, event);
  • Save as inscreen.hyperesources
  • Upload .hyperesources folder to a CDN directory
